# 04 — Judging Score Maximizer

## 1) Commercial ROI (30%)
- Show direct savings: reduce migration discovery from weeks to hours.
- Quantify risk reduction through automated checks + audit trail.
- Present target ICP and pricing tiers.

## 2) Innovation & UX (30%)
- Multi-agent workflow with explainable decisions.
- Clean dashboard with migration timeline and risk badges.
- Human approval checkpoints for confidence.

## 3) Technical rigor (20%)
- Proper service boundaries and async orchestration.
- Test/lint/security gates integrated in pipeline.
- Production deployment on GCP with logs and metrics.

## 4) Presentation/Demo (20%)
- Use one storyline: “legacy repo in, cloud-ready service out.”
- Show before/after metrics visually.
- End with running production URL and rollback confidence.

## Demo evidence checklist
- Architecture diagram.
- Live pipeline run.
- Auto-generated migration report.
- PR diff with rationale.
- Deployed endpoint on GCP.
