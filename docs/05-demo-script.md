# 05 — Demo Script (5–7 minutes)

## Minute 0–1: Problem and opportunity
“Legacy migration is expensive, risky, and slow. Our Agentic Migration Factory converts it into an auditable, AI-driven pipeline.”

## Minute 1–2: Product overview
- Create project.
- Connect/import repo.
- Trigger analysis.
- View migration plan.

## Minute 2–4: Live execution
- Run migration task.
- Show generated patch/PR notes.
- Show validation report (tests, lint, security).

## Minute 4–5: Deployment proof
- Deploy migrated service to Cloud Run.
- Open live URL and health endpoint.
- Show logs/metrics dashboard.

## Minute 5–6: Business case
- Time saved.
- Risk managed.
- Team productivity impact.

## Minute 6–7: Roadmap
- Add language packs.
- Add enterprise controls (SSO, policy rules).
- Add portfolio migration mode for multi-repo programs.
